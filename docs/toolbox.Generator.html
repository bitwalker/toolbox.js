Class: Generator

src/toolbox. Generator

new Generator(generator, seed)

Generates a value from a function using a provided seed value. Every subsequent call to next uses the last generated value as the new seed. This can be used to generate an infinite sequence which can be iterated over using next as if it was a regular enumerable like an array.

Parameters:
Name Type Description
generator function

The generating function
seed any

The first value to send to the generator function
