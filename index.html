Toolbox.js

A general purpose javascript utility library. Contains everything you need, and most anything worth having (other than DOM manipulation).

View project onGitHub

Why Toolkit.js?

Fair question. I've been a fond user of Underscore.js, as well as underscore.string, for quite some time. I found Underscore generally lacking in the functional programming tools department, which is what I initially started putting this library together for. I figured that I'd rather have one dependency that fulfills my needs than two, so I also re-implemented the collections functions that I used Underscore for. Over time this evolved into adding little utility functions that I thought would likely be needed again, and the toolkit was born.

I also wanted this to be reusable across platforms (Node.js, the browser), as well as allow for the use of module loaders like require.js (hence the AMD module compatibility). It is an additional goal to keep the minified version of this project as small as possible, while still providing as much utility as possible. I build new functions by reusing existing components where possible, and in that way I hope to save on file size, as well as keep thing maintainable in the long run.

How do I use it?

You can either add a <script> tag to your page, or require it in Node. The un-minified source is well commented, but if it's more readable, the full html documentation is available in the /docs directory of the repository.
